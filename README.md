# nmp
This folder contains data sets related to NMP experiments performed at the MMLab. The files are:

nmp-data-set.xlsx: An Excel file containing results from NMP experiments organized by Konstantinos Tsioutas. Each sheet contains data from one pair of musicians, performing with 10 different delays. For each delay setting, the file provides a set of audio features, the measured tempo and a set of subjective evaluation variables.
